# 快速熟悉项目-入手指南

第1步：阅读Introduction.md文件，了解项目风格与内容，文字描述本次高通版XLeRobot的功能简介，images_intro文件夹里存放的是介绍文档的用图，可不用查看。

第2步：查看Principle文件里的原理介绍，可快速浏览以帮助开发者对原理基础有一个大致了解。

第3步：查看Hardware_List文件，清单罗列了项目所需的全部硬件物料，并附上链接，开发者可自行采购材料。

第4步：查看3D_Printing_Models文件里的3D模型，该文件存放了这个项目需要3D打印的所有模型，开发者可自行打印部件。

第5步：浏览Assembly_Steps_and_Structure_Diagram文件夹，内含有机械臂、底盘、顶部支座的组装方法，并含有机械臂安装教学视频，有结构图帮助开发者更直观地进行硬件安装。

第6步：查看Wiring_Diagram文件夹下的接线图，学习如何接线，理解如何连通所有部件。

第7步：硬件组装好并接线连通后，可阅览Set_Servo_ID文件夹，内容讲述如何对所有舵机进行ID设置和编号顺序。

第8步：硬件的所有工作已完成，接下来可查看Code_Resources里的内容，这部分是对软件层面的使用说明，内含有软件部分的README.md文件，介绍了机器人软件实现的详细方法。

（可选）可查看Demonstration.mp4文件，这是项目的演示视频。
